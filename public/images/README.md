# Sangeetha Lahari Events - Image Assets Guide

Place your high-resolution luxury event images in this directory using the filenames specified below. The homepage components will automatically load them.

## Required Image Assets

| Filename | Description | Suggested Aspect Ratio | Recommended Resolution |
|---|---|---|---|
| `hero-bg.jpg` | Background image for the full-screen cinematic hero. | 16:9 (Landscape) | 1920 × 1080 |
| `service-wedding.jpg` | Thumbnail image for the Wedding Events service card. | 4:3 or 16:10 | 800 × 600 |
| `service-corporate.jpg` | Thumbnail image for the Corporate Events service card. | 4:3 or 16:10 | 800 × 600 |
| `service-social.jpg` | Thumbnail image for the Social Celebrations service card. | 4:3 or 16:10 | 800 × 600 |
| `about-ceo.jpg` | Portrait photo of the Chief Organizer / CEO (Mr. Suresh). | 3:4 (Portrait) | 600 × 800 |
| `gallery-wedding-1.jpg` | Wedding ceremony, grand Mandap floral setup. | Masonry fit (mixed) | 1200 × 800 (or vice versa) |
| `gallery-reception.jpg` | Ballroom reception setup with luxury chandeliers. | Masonry fit | 1200 × 800 |
| `gallery-corporate.jpg` | Corporate event stage or gala seating layout. | Masonry fit | 1200 × 800 |
| `gallery-birthday.jpg` | Outdoor birthday or theme celebration with decor. | Masonry fit | 1200 × 800 |
| `gallery-stage.jpg` | Grand stage decoration closeup. | Masonry fit | 1200 × 800 |
| `gallery-haldi.jpg` | Vibrant Haldi setup with marigold flowers. | Masonry fit | 1200 × 800 |
| `gallery-mehendi.jpg` | Mehendi function details or seating area. | Masonry fit | 1200 × 800 |
| `cta-bg.jpg` | Background cover image for the pre-footer Consultation CTA. | 16:9 (Landscape) | 1600 × 900 |

## Note on Fallbacks
If these image files are missing, the website will display elegant SVG-based luxury patterned gradients and descriptive placeholder badges with clean borders to maintain the visual hierarchy and structure.
